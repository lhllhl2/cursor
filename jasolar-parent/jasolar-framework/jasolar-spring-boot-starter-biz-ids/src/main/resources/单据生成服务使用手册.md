# 单据号生成服务开发使用手册
## 1. 概述
  本手册旨在指导开发者如何使用和扩展单据号生成器。
## 2. 主要组件
### 2.1 抽象单据号生成器
> 提供生成序列号和单据号的基本实现
>
> ```java  
> public abstract class AbstractBizIdGenerator implements IBizIdGenerator {  
> @Resource  
> private RedisSerialNumberGenerator redisSerialNumberGenerator;
>
>    public long generateSerialNumber(String bizType) {  
>         return redisSerialNumberGenerator.generateSerialNumber(bizType);  
>     }  
>
>     public String generateBizId(@Nullable String bizType, @Nullable Integer length, @Nullable String prefix, @Nullable String suffix) {  
>        long sequence = this.generateSerialNumber(bizType);  
>         int len = length == null ? 0 : length;  
>         String sequenceString = String.format("%0" + len + "d", sequence);  
>         StringBuilder bizId = new StringBuilder();  
>         if (StringUtils.isNotBlank(prefix)) {  
>             bizId.append(prefix);  
>         }  
>         bizId.append(sequenceString);  
>         if (StringUtils.isNotBlank(suffix)) {  
>             bizId.append(suffix);  
>         }  
>         return bizId.toString();  
>     }  
> }
> ```  

### 2.2 业务枚举类
> 实现 IBizType 接口，提供具体的业务类型及其相关属性（前缀、后缀、流水号位数）,根据业务不同，重写对应的前缀，后缀，以及流水号长度数据，实现单据号的定制化。
> 以下是一个示例实现：
>
> ```java  
> public enum SupplierBizTypeEnum implements IBizType {  
> 
>   SUPPLIER("SUPPLIER", 5, "xxx", "xxx");  
> 
>
>     /**
>      * 业务类型(唯一）
>      */
>     private final String bizType;
>
>     /**
>      * 序列长度
>      */
>     private final Integer sequenceLength;
>
>     /**
>      * 前缀
>      */
>     private final String prefix;
>
>     /**
>     * 后缀
>     */
>     private final String suffix;
>
>     SupplierBizTypeEnum(String bizType, Integer sequenceLength, String prefix, String suffix) {
>         this.bizType = bizType;
>         this.sequenceLength = sequenceLength;
>         this.prefix = prefix;
>         this.suffix = suffix;
>     }
>
>     @Override
>     public String getBizType() {
>         return bizType;
>     }
>
>     @Override
>     public Integer getSequenceLength() {
>         return this.sequenceLength;
>     }
>
>     @Override
>     public String getPrefix() {
>         return this.prefix  ;
>     }
>
>     @Override
>     public String getSuffix() {
>         return this.suffix;
>     }
> }
> ```  

### 2.3  单据号生成工具类
> 提供静态方法根据业务类型生成单据号
>
> ```java  
> public class BizIdGeneratorUtils {  
>    public static String generateBizId(IBizType bizType) {  
>        if (null == bizType) {  
>            return null;  
>        }  
>        IBizIdGenerator bizIdGenerator = SpringUtils.getBean(com.fiifoxconn.mis.framework.ids.generator.DefaultGenerator.class);  
>        if (null != bizIdGenerator) {  
>            return bizIdGenerator.generateBizId(bizType.getBizType(), bizType.getSequenceLength(), bizType.getPrefix(), bizType.getSuffix());  
>        }  
>        return null;  
>    }  
>}
> ```  

## 3. 使用步骤
### 3.1 引入依赖
> 确保在项目中引入单据号生成模块依赖
>
> ```java
> <!-- 业务单据号生成模块 -->
> <dependency>
>   <groupId>com.fiifoxconn.mis</groupId>
>   <artifactId>fiifoxconn-spring-boot-starter-biz-ids</artifactId>
> </dependency>
> ```   


### 3.2 生成方式
> 在需要生成业务 ID 的地方，调用 BizIdGeneratorUtils.generateBizId(...) 方法：
>
> ```java
> String bizId = BizIdGeneratorUtils.generateBizId(SupplierBizTypeEnum.SUPPLIER);  
> System.out.println("生成的单据号：" + bizId);
> ```       

### 3.3 扩展业务类型
> 如需增加新的业务类型，在各自模块添加相应的枚举并实现 IBizType 接口。
>
> ```java
> public enum NewBizTypeEnum implements IBizType {  
>       NEW_TYPE("NEW_TYPE", 10, "newPrefix", "newSuffix");  
>       // 其他属性和方法...  
> }
> ```
> 注意：   
> 1、在使用过程中要确保bizType唯一，否则会出现重复</br>
> 2、通过实现IBizType接口，重写获取序列长度、获取前缀、获取后缀接口实现单据号的定制
>
## 4. 注意事项
   1、 确保 Redis、数据库、infra服务正常运行，以避免单据号生成失败。</br>
   2、 确保 定时任务（refreshSnJob）正常运行，以防止在redis宕机的情况下，当日的单据号数据能及时同步到DB。 </br>
   3、 在生产环境中，合理配置序列号长度和前后缀，以满足业务需求。</br>
